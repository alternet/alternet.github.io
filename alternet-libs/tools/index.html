Summary

Published version of this page available HERE

Alternet Tools include discovery service tools, concurrent and locking tools, byte code generator and more.

Documentation

Alternet Tools contains mainly :

  • Properties binder and classes generator : you have Java properties files, you want to populate a class with the properties ; you also want to generate the class to populate ; you want a Maven plugin for that
  • Discovery Service : you have an interface and various implementations that you want to bind by name.
  • Generator : you have an interface, you want to generate an instance of it.
  • Alternet Tools : you want other tools and utilities (concurrent and locking tools, char range, type handling, etc)
