There are several ways you can contribute to the betterment of the project.

Report Issues

Issue reporting is central to OSS projects. If the issue isn’t reported, we can’t fix it. Please report any bugs, feature, and/or improvement requests on GitHub.

Submit Patches

Have a fix or a feature you'd like to share? Open an issue and link your patch with details on what you're resolving and how you've resolving it.

Documentation

Good documentation is an important aspect of any software project. We've taken several strides in improving our documentation, but there's always room for improvement. If you see anything missing or an area that could be improved, let us know or submit a patch.

Write a Blog

Are you using Alternet Libs ? We'd love to hear how you're using it in your projects. Write a blog and drop us a note.

Code Review

More eyes on the code means better quality. Review our code for style issues, potential issues, or glaring bugs. Anything helps!
